1. 产品简介

ST Deck Agent 帮助您在浏览器中根据文字需求与参考文档，自动生成符合 ST PPT 品牌规范的 .pptx，并提供逐页预览与下载。

当前主能力：Deck 生成（一键成稿、可选对话模式 Beta、在线编辑）。

2. 访问地址与访问口令

  1. 向管理员获取 Cloud Run 服务 URL（服务名通常为 st-deck-agent）。
  2. 若页面右上角出现 访问口令，请输入管理员发放的 ACCESS_TOKEN；浏览器会记住口令，后续请求自动携带。
  3. 预览图与下载也可在链接后加 ?access_token=…（无法自定义请求头时使用）。

3. 使用须知

每个浏览器会话首次打开时须确认弹窗，内容包括：

  • 遵守 STMicroelectronics《行为准则》；
  • 不得上传或引用涉密、保密、出口管制或未公开内部资料；
  • 使用前完成《人工智能法律合规意识》（Legal Awareness in Artificial Intelligence）培训；违规后果由用户承担。

顶栏可切换界面语言（中文 / EN / 日本語），弹窗文案随之切换。

4. 界面概览

左侧需求、页数、幻灯片语言、参考文档、对话模式、生成按钮
右侧工作台：待命说明 / 运行日志 / 预览与下载 / 编辑或对话

幻灯片语言 与 界面语言 相互独立：前者决定 PPT 内文字语言。

5. 生成 PPT（推荐）

撰写需求

说明受众、主题、必须包含的数据；可附 URL 供截图。点击「试试」示例可快速填入。快捷键：Ctrl+Enter（Mac：⌘+Enter）。

页数

建议：同一主题先生成多页，再通过在线编辑或本机 PPT 合并，内容更丰富。仅 1 页时系统按「高管摘要单页」排版，避免三页故事挤在一页。

参考文档

PDF、Word、PPT、Excel、图片等。默认限制：单文件 10 MB、合计 30 MB、最多 5 个文件。

运行中

可点「停止」中断。完成后可下载并查看每页 PNG 预览。复杂任务可能触发约 9 分钟服务端超时，请减少页数或简化需求。

6. 对话模式（Beta）

开启后首轮只出大纲与追问，不会立即生成文件。在「继续对话」中补充信息，明确「确认」或 OK 后才开始 BUILD。

  • 需服务部署为单实例（max-instances=1），否则多轮对话可能失败；
  • 实例重启或超过 24 小时会话过期后上下文丢失；
  • 对话完成后不提供「在线编辑」，请继续在对话中说明修改。

7. 在线编辑

仅在一键生成成功且未开对话模式时可用。用自然语言描述修改，点击「应用修改」；Agent 更新 build.py 并重新渲染预览。

8. 下载与预览

  • 下载名一般为「主题-日期.pptx」；
  • 网页预览使用 Liberation Sans，本机 PowerPoint 以 Arial 显示，细微差异正常；
  • 会话约 24 小时后删除，请尽快下载；
  • 对外材料请自行核对 closing / 商标等合规页。

9. 使用限制（默认）

每 IP 每分钟请求10
最大页数6
会话保留24 小时
单次运行超时约 540 秒

10. 常见问题

Unauthorized？ 检查访问口令。

生成失败？ 简化需求、减少页数；联系管理员查日志。

对话无响应？ 可能多实例或会话过期，改用手动一键模式。

预览与 PPT 不一致？ 以本机 PowerPoint 为准。

部署与故障排查请参阅仓库 README。

11. Deck 焕新（规划中）

未来将支持上传自有 .pptx，仅做版式与品牌视觉标准化、不改文案。默认环境尚未开放该 Tab。产品说明见仓库 docs/PRD-DECK-REFRESH.md。
